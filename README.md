Ini adalah starter-template untuk membangun aplikasi SaaS yang skalabel menggunakan Next.js dan Supabase self-hosted.

## Features

- Page Templates
  - Marketing landing
  - Pricing page
  - FAQs
  - Changelog
  - Legal Compliance
  - Custom
- CMS (custom atau integrasi ke cms populer seperti wordpress, dll)
- Translations (i18n)
- Authentication
- Dashboard
- Multi-Tenancy (Organizations/Teams)
- Super Admin
- Billing
- AI
  - MCP Server
  - More _(Brainstorm later)_
- Settings
  - Profile
  - System (themes, lang, analytic, storage, etc)
  - Email (config for e.g. smtp/resend, sending, template, etc)
  - Notification
- Documentation (for developer)

## Note

Saya sudah menginstall next.js beserta semua tumpukan teknologi yang dibutuhkan.
